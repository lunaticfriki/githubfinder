# Github finder

## Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

### Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

### Run tests

```bash
npm run test
# or
yarn test
# or
pnpm test
```

### Run tests coverage

```bash
npm run test:coverge
# or
yarn test:coverge
# or
pnpm test:coverge
```

### Run tests UI

(you can see a UI with graphs, reports and code of each component test)

```bash
npm run test:ui
# or
yarn test:ui
# or
pnpm test:ui
```
